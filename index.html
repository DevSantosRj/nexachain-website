Sobre o Projeto

Conheça a NEXACHAIN, uma blockchain inovadora que resolve o trilema da blockchain através de um consenso híbrido e um modelo económico deflacionário.

Uma Nova Geração de Blockchain Híbrida

A NEXACHAIN é uma blockchain descentralizada que combina a eficiência e as funcionalidades avançadas do XRP Ledger (XRPL) com a robustez e descentralização do Bitcoin.

Consenso Híbrido

Combinação de Proof of Work (PoW) para seleção de líderes de bloco com o Ripple Protocol Consensus Algorithm (RPCA) para validação eficiente.

Modelo Deflacionário

Supply inicial de 100 milhões de tokens NEX com mecanismos de queima de taxas e halvings anuais.

Alta Performance

~1500 transações por segundo (TPS) com confirmações em 3-5 segundos.

Tecnologia Inovadora

A NEXACHAIN resolve o trilema da blockchain através de uma arquitetura técnica inovadora que combina o melhor de duas tecnologias comprovadas.

Consenso Híbrido PoW + RPCA

A NEXACHAIN utiliza um mecanismo de consenso híbrido que combina o melhor de dois mundos: a descentralização do Proof of Work (PoW) com a eficiência do Ripple Protocol Consensus Algorithm (RPCA).

Proof of Work (PoW)

Qualquer nó pode minerar blocos resolvendo um desafio SHA-256, garantindo descentralização ao estilo Bitcoin. A dificuldade é ajustada dinamicamente para manter blocos a cada ~3-5 segundos.

Ripple Protocol Consensus Algorithm (RPCA)

Validadores confiáveis verificam propostas de blocos, garantindo consenso rápido e sem forks. A lista de validadores é aberta, permitindo que nós com reputação se juntem.

Eficiência Energética

O PoW é usado apenas para propor blocos, não para validação completa, reduzindo o consumo energético em comparação com o Bitcoin (~0,1-1 TWh/ano estimado para 1000 nós vs. 150 TWh/ano do BTC).

Arquitetura Técnica

A NEXACHAIN é baseada no servidor open-source rippled do XRP Ledger, com modificações para suportar o consenso híbrido e o modelo económico deflacionário.

bool proofOfWork(const std::string& blockData, int difficulty) {
  std::string target(difficulty, '0');
  uint64_t nonce = 0;
  while (true) {
    std::string hash = sha256(blockData + std::to_string(nonce));
    if (hash.substr(0, difficulty) == target)
      return true;
    nonce++;
  }
}

Modelo Deflacionário

Queima de Taxas

Cada transação queima uma taxa base (0,0001 NEX). Com 1000 TPS, aproximadamente 3,15M NEX são queimados por ano.

Halvings de Recompensa

Recompensa inicial: 10 NEX por bloco. Halving a cada 1 ano: 10 NEX → 5 NEX → 2,5 NEX, etc.

Queima Adicional

0,1% do valor de transações na DEX é queimado. Criação de NFTs e execução de smart contracts podem incluir queima opcional.

Projeção de Supply

Após aproximadamente 4 anos (2 halvings), a queima deve superar a emissão, tornando o NEX deflacionário.
